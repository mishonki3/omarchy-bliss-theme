## Install with Bash:

omarchy-theme-install https://github.com/mishonki3/omarchy-serenity-theme.git


## Install with GUI:
Copy https://github.com/mishonki3/omarchy-serenity-theme
Click SUPER + ALT + SPACE
Click Install
Click Theme
Paste the link
